| –û–ø–µ—Ä–∞—Ü–∏—è | –û–ø–µ—Ä–∞—Ç–æ—Ä | –ú–µ—Ç–æ–¥ | –†–µ–∑—É–ª—å—Ç–∞—Ç | –ú–Ω–µ–º–æ–Ω–∏–∫–∞ |

| –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ | `\|` | `union()` | `{1, 2, 3, 4, 5}` | "\| = –¢–†–£–ë–ê –°–û–ï–î–ò–ù–Ø–ï–¢" |
| –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ | `&` | `intersection()` | `{3}` | "& = –û–ë–©–ò–ô –£–ó–ï–õ" |

### üß† –í–∏–∑—É–∞–ª—å–Ω–∞—è –º–Ω–µ–º–æ–Ω–∏–∫–∞:
- `|` - —Ç—Ä—É–±–∞ —Å–æ–µ–¥–∏–Ω—è–µ—Ç –≤—Å—ë
- `&` - –∞–º–ø–µ—Ä—Å–∞–Ω–¥ –∫–∞–∫ –æ–±—â–∏–π —É–∑–µ–ª
- `-` - –º–∏–Ω—É—Å —É–±–∏—Ä–∞–µ—Ç
- `^` - –∫—Ä—ã—à–∞ –ø–æ–∫—Ä—ã–≤–∞–µ—Ç —Ç–æ–ª—å–∫–æ —É–Ω–∏–∫–∞–ª—å–Ω–æ–µ

## üîç –ü—Ä–æ–≤–µ—Ä–∫–∏ –∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è

| –ü—Ä–æ–≤–µ—Ä–∫–∞ | –û–ø–µ—Ä–∞—Ç–æ—Ä | –ú–µ—Ç–æ–¥ | –ú–Ω–µ–º–æ–Ω–∏–∫–∞ |
|----------|----------|-------|-----------|
| –ü–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ | `<=` | `issubset()` | "<= –ú–ï–ù–¨–®–ï –ò–õ–ò –í–ù–£–¢–†–ò" |

| –í—Ö–æ–∂–¥–µ–Ω–∏–µ | `in` | - | "IN = –í–ù–£–¢–†–ò" |

